# Proposition de scripts — MGGT1103

Objectif : créer des raccourcis Bash utiles dans le cadre du cours MGGT1103.

Ces scripts permettent de gagner du temps, vérifier un rendu, afficher rapidement les résultats importants et automatiser certaines tâches répétitives du laboratoire.

## Scripts proposés

### Présentation

- `afficher-resume-tp1.sh`  
  Affiche un résumé rapide du TP1 : environnement DevOps, Git, Vagrant, captures et GitHub.

### Vérification

- `verifier-tp1.sh`  
  Vérifie les fichiers essentiels du TP1, les captures, l’état Git et les derniers commits.

- `tableau-bord-mggt1103.sh`  
  Affiche une vue globale du projet : Git, TP1, captures, Vagrantfile, scripts et rapports.

- `verifier-rendu-mggt1103.sh`  
  Script générique utilisable pour vérifier un rendu MGGT1103 dans n’importe quel dépôt étudiant.

### Git et GitHub

- `etat-git-projet.sh`  
  Affiche la branche active, l’état Git, les derniers commits et le dépôt distant.

### Vagrant

- `commandes-vagrant-lab.sh`  
  Affiche les commandes Vagrant importantes à utiliser dans PowerShell Windows.

### Linux

- `diagnostic-systeme-linux.sh`  
  Affiche un diagnostic rapide : utilisateur, machine, noyau, mémoire, disque, réseau et processus.

- `gestion-permissions-linux.sh`  
  Affiche les permissions, propriétaires, groupes et commandes utiles pour gérer les droits Linux.

### Travaux pratiques

- `preparer-rendu-tp1.sh`  
  Prépare automatiquement les dossiers de rendu du TP1 et copie les fichiers importants.

- `preparer-rendu-tp2.sh`  
  Prépare automatiquement les dossiers de rendu du TP2 : rapport, captures et script de sauvegarde.

## Utilité pédagogique

Ces scripts ne remplacent pas l’apprentissage des commandes.  
Ils servent de raccourcis professionnels pour vérifier, présenter et organiser le travail réalisé dans le cadre du cours.
