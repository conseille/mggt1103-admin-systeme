## Présentation du dépôt

Ce dépôt contient l’organisation complète de mes travaux pratiques du cours MGGT1103.

Il a pour objectif de présenter clairement les exercices réalisés, les rapports produits, les captures de preuve, les scripts utiles et les outils de présentation liés à mon apprentissage en administration système, virtualisation, automatisation et DevOps.

L’organisation du dépôt a été simplifiée afin de garder uniquement les éléments importants pour le suivi du cours.

## Organisation principale

```text
MGGT1103_Cours_Adm_Systeme_MUPINI-KABWE-Albert/
├── 01-travaux-pratiques/
```


## 02 — Laboratoire d’automatisation

Le dossier `02-laboratoire-automatisation/` est réservé aux scripts et outils d’automatisation utiles au cours.

Les éléments placés dans ce dossier auront un rôle clair :

- automatiser une tâche répétitive ;
- vérifier un environnement ;
- faciliter un travail pratique ;
- servir de base réutilisable pour les prochaines séances.

## 03 — Outils de présentation

Le dossier `03-outils-de-presentation/` contient les outils utilisés pour présenter le travail de manière claire.

Ce dossier contiendra des interfaces, des tableaux de bord ou des pages de présentation liées aux travaux réalisés.
