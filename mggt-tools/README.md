# mggt-tools

`mggt-tools` est un ensemble d’assistants terminal pour aider à vérifier,
présenter et explorer les travaux pratiques du cours MGGT1103.

## Outils disponibles

- `mggt-assistant` : assistant professeur principal ;
- `tpnav` : navigation interactive dans les TP ;
- `gitnav` : assistant Git complet ;
- `vagrantnav` : assistant Vagrant / VirtualBox.

## Installation utilisateur

Depuis la racine du dépôt contenant `mggt-tools` :

```bash
bash mggt-tools/install/install.sh
source ~/.bashrc
